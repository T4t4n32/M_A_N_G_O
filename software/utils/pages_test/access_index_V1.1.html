Objetivos del Proyecto

M.A.N.G.O. nace de la necesidad de obtener datos ambientales precisos y continuos con una infraestructura de bajo costo y alta fiabilidad. Buscamos democratizar la monitorización oceánica.

  • Recolección autónoma de datos de nivel y temperatura del agua.
  • Implementación de un servidor virtualizado (Proxmox) para gestión y análisis centralizados.
  • Desarrollo de un dashboard web para visualización de datos en tiempo real (evidencia de construcción).
  • Garantizar la escalabilidad y el mantenimiento a largo plazo del sistema.

Evidencia de Construcción (Tech Stack)

La solidez del proyecto M.A.N.G.O. se basa en tecnologías robustas, desde el hardware de los sensores hasta la infraestructura de virtualización.

Infraestructura de Servidor:

  • **Virtualización:** Proxmox VE sobre Debian 12.
  • **Base de Datos:** PostgreSQL / InfluxDB (para series de tiempo).
  • **Código:** Python (backend) y HTML/CSS/JS (frontend).
